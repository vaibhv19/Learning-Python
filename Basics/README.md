# Python Basics

This repository is a learning resource for Python fundamentals, organized by chapter. Each chapter contains example files and practice problems that cover core Python concepts from variables and data types to functions and recursion.

## Repository Structure

- `chapter 1/` - Basic Python syntax and simple examples
- `chapter 2/` - Variables, data types, operators, and input handling
- `chapter 3/` - String operations, slicing, and string functions
- `chapter 4/` - Lists, tuples, and collection basics
- `chapter 5/` - Dictionaries, sets, and set operations
- `chapter 6/` - Conditional statements and branching logic
- `chapter 7/` - Loop constructs and iteration patterns
- `chapter 8/` - Functions, arguments, default values, and recursion

Each chapter folder also contains a `PS/` subfolder with practice problems and exercises for hands-on learning.

## How to Use

1. Open the chapter folder for the topic you want to study.
2. Read the example Python files to understand the concept.
3. Try the practice problems in the `PS/` folder.
4. Run Python files using a command like:

```bash
python "chapter 2/1_variables.py"
```

## Notes

- The repository is intended for beginners learning Python programming.
- Example files use basic Python syntax and should run with Python 3.
- Practice files are labeled numerically and are useful for self-study.

## Getting Started

If you are new to Python:

1. Start with `chapter 1/` to get familiar with syntax.
2. Continue to `chapter 2/` and `chapter 3/` for data types and string operations.
3. Use practice problems in each chapter to reinforce concepts.
